Description: Introduction:
The School of Infocomm (SOI) Learning Portal is an online catalogue of various lessons ranging from coding to UI/UX design lessons that provides users with an easy way to sign up for various courses that would best fit their interest and learning needs.

Key Features:
- Carousel featuring various courses as well as the duration of the course
- Allows users to save their favourite courses
- Search bar, with auto complete functionality
- Toggle feature indicating online/offline status

Development:
This website was built using HTML, CSS and Javascript.
