Category: App Development

Language: Java

Year: 2023

Description: Introduction:
Developed a simple movie list app that allows users to add/delete/edit/update movies as well as filter out PG-13 movies.

Key Features:
- Allows user to input a new movie
- Allows user to edit/delete entries made previously

Development:
This app was developed in Android Studio using Java. In this app, users are allowed to input movie details such as the name of the movie, genre, year of release, and rating for each movie. Upon clicking on the 'Insert' button, a dialog box appears, prompting users to confirm their input. If user clicks on 'Cancel', the input will be discarded, and when user clicks on 'Insert', a toast message appears, confirming the successful addition of the movie. The app also features a 'Show List' button, enabling users ot view all the added movies in a separate activity. Within this activity, users have the ability to filter movies based on the ratings, specifically allowing them to display movies with a PG-13 rating. Additionally, the list of movies in the new activity provides interactive functionality. Users can click on individual movies to access options such as 'Update' and 'Delete'. The 'Update' option allows users to modify details of the movie, while the 'Delete' option removes the selected movie from the list, providing users with comprehensive control over the song database within the app.
