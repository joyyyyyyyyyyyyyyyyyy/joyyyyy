Category: App Development

Language: Java

Year: 2023

Description: Introduction:
This app allows users to book/reserve a spot in the restaurant with their information such as name, contact number, number of pax for any time and day. It also allows users to choose the type of seating area they prefer, such as smoking or non-smoking area.

Key Features:
- Allows user to make a booking for any pax at any time of the day
- Allows user to select type of seating preferred

Development:
This app was developed in Android Studio using Java. It also utilises the usage of essential components such as DatePickers, TimePickers to allow the app to function as intended. Users are required to input their name, mobile number, number of pax, as well as select if they preferred a smoking or non-smoking area before clicking on submit. Upon clicking submit, a dialog box displaying the details of reservation will be displayed.
